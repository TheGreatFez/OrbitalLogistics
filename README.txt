Body.m: Body Class definition file
Shuttle.m: Shuttle Class definition file
OrbitSim.m: Script to run the orbit simulation
	- Creates 2 Body's (Earth and Mars) at given Mean Anomalies
	- Initial Time is 0sec
	- Creates 1 Shuttle with the origin at Earth
	- Initial Mars position is set to create an intercept between the Shuttle and Mars (Hohmann Transfer)
